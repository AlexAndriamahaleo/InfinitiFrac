# Infiniti'Frac

Project for the purpose of study

### Run application

* Clone the project or download .zip file

* Move to directory
    
    `~ $ cd Infiniti'Frac`
    
* Build executable
    
    `~/Infiniti'Frac $ make all`
    
* Run
    
    `~/Infiniti'Frac $ ./logiciel`
